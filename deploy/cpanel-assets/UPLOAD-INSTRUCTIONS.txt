cPanel upload instructions
=======================

IMPORTANT: Drag-and-drop onto the cPanel file list does NOT work.
Use the Upload button, or see deploy/CPANEL-DEPLOY.md for full guide.

Quick steps:
1. cPanel > File Manager
2. Open public/assets inside your app root (where package.json lives)
3. Click Upload (top toolbar) — do NOT drag onto the file list
4. Upload assets-part-01-of-XX.zip, then 02, 03... one by one
5. Select each zip > Extract
6. Files must end up directly in public/assets/
7. Delete zip files after extraction
8. Verify: npm run verify:assets -- https://YOUR_DOMAIN
